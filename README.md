# Payment Collection System

A fun, interactive web application that simulates a payment collection system with animated modals and TikTok-style chat interface.

## Features

- Animated payment collection button
- Interactive modal system
- Simulated phone call animation
- TikTok-style chat interface with:
  - Gradient text animations
  - Verified badge
  - Floating hearts animation
  - Interactive reactions
- Responsive design for various screen sizes

## Technologies Used

- HTML5
- CSS3 (with animations and transitions)
- JavaScript (vanilla)
- Custom animations and effects

## How to Use

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start the development server
4. Open the provided URL in your browser

## Project Structure

- `killer_ntua/killer_ntua.html` - Main application file
- `public/` - Static assets (images, etc.)
